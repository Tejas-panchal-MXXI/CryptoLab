# CryptoLab

CryptoLab is an interactive cryptography learning platform designed to help users understand and apply fundamental cryptographic concepts through practical tools and visualizations.

## Features
- Real-time password entropy analysis and visualization
- Classic cipher tools including substitution and transposition ciphers
- Hash generator supporting multiple hashing algorithms
- User-friendly tabbed interface for navigating different cryptography tools
- Security tips and best practices for password creation
